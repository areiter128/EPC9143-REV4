Detailed Description

Declaration of phase-current feedback properties.

Description: In this section the phase-current feedback signal scaling, gain, valid signal limits and nominal operating point is specified. Physical values are used to define values. Macros are used to convert given physical values into binary (integer) number to be written into SFRs and variables.
