enum FLT_COMPARE_TYPE_e

This data structure is comprise of fault comparison types.

Enumerator
FLTCMP_NONE 

No comparison type has been defined (fault object will be ignored)

FLTCMP_GREATER_THAN 

Check for condition: SOURCE > trip level.

FLTCMP_LESS_THAN 

Check for condition: SOURCE < trip level.

FLTCMP_IS_EQUAL 

Check for condition: SOURCE = trip level.

FLTCMP_IS_NOT_EQUAL 

Check for condition: SOURCE != trip level.

FLTCMP_BETWEEN 

Check for condition: (reset_level < SOURCE) && (SOURCE < trip_level)

FLTCMP_OUTSIDE 

Check for condition: (SOURCE < reset_level) || (trip_level < SOURCE)
