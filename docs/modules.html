 Power Control Handler EnumerationsThis enumeration is listing all defined states supported by the power controller state-machine. The state machine handles the initialization of the power controller, stand-by, start up procedure including Power-On-Delay, Ramp-Up and Power Good Delay until it ends up in a continuous operating state. When reference values are changed while running, the state machine will tune into the new reference values using the slew rates defined for the startup phase
