Detailed Description

Global defines for Buck Converter Power Control parameters.

Description This section is used to define hardware specific parameters such as output voltage dividers, reference levels or feedback gains. Pre-compiler macros are used to translate physical
values into binary (integer) numbers to be written to SFRs
